\doxysubsection{Detailed Description}
Write trajectories of any type as delimited text. 

This class writes subclasses of \doxylink{classtracktable_1_1_trajectory}{tracktable\+::\+Trajectory} (including the domain classes) to a stream as delimited text. It will write one trajectory per line. The resulting file will contain enough header information to reconstruct the trajectory exactly as long as the user asks for the correct class.
