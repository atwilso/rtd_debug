Create distance geometry signature with samples by distance. 

This function computes the multilevel distance geometry for a given trajectory. Each level {\itshape d} approximates the input trajectory with {\itshape d} equal-\/length line segments. The distance geometry values for that level are the lengths of all {\itshape d} line segments, normalized to lie between 0 and 1.

The D-\/level distance geometry for a curve will result in (D \texorpdfstring{$\ast$}{*} (D+1)) / 2 separate values.

This implementation creates the endpoints of the line segments by sampling the trajectory at fractions of total distance traveled. To sample by total duration, use \doxylink{namespacetracktable_ad311e0eda396d74a316f442c0d740e64}{distance\+\_\+geometry\+\_\+by\+\_\+time()}.

Retrieve a timestamp property. 

Retrieve a timestamp-\/valued property from the map.

This accessor will let you retrieve the value of a timestamp property. The catch is that we don\textquotesingle{}t know whether or not the property is there to begin with. If it isn\textquotesingle{}t there then we can\textquotesingle{}t return anything sensible.

We deal with this by letting you pass in an optional pointer to a boolean. We will set its value to true or false depending on whether or not we found the property you wanted. If it is true, the return value is guaranteed to be whatever is in the map. If it is false, the return value will be uninitialized.

\begin{DoxyNote}{Note}
For the purposes of this function, a property that is present but that has the wrong type is the same as a property that is not present in the map.
\end{DoxyNote}
