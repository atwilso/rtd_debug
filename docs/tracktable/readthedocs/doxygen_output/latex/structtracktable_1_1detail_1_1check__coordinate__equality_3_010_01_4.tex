\doxysubsection{Detailed Description}
Helper object for checking equality between points. 

Compare two points for equality by comparing all of their coordinates.

You will definitely not need to instantiate this object directly. This is the base case for the iteration over the coordinates of a point that causes the template instantiation to stop after looping over all of the dimensions at compile time.
