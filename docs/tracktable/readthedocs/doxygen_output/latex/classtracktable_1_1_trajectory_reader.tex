\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename \mbox{\hyperlink{classtracktable_1_1_trajectory}{TrajectoryT}}$>$\newline
class tracktable\+::\+Trajectory\+Reader$<$ Trajectory\+T $>$}
Read trajectories from files. 

This reader wraps the following pipeline\+:


\begin{DoxyItemize}
\item Read lines from a text file
\item Skip any lines that begin with a designated comment character (\textquotesingle{}\#\textquotesingle{} by default)
\item Tokenize each line using specified delimiters (whitespace by default)
\item Create a trajectory (user-\/specified type) from each tokenized line
\item Return the resulting points via a C++ iterator
\end{DoxyItemize}

You will use \doxylink{classtracktable_1_1_trajectory_reader_aa8019fa8c541a4880b0c8ab03787238b}{set\+\_\+input()} to supply an input stream, \doxylink{classtracktable_1_1_trajectory_reader_aa7fe312ba51efb87668caafa4cec6ab3}{set\+\_\+comment\+\_\+character()} to configure which lines to skip, \doxylink{classtracktable_1_1_trajectory_reader_ad8b8912bc589d9cfb809e47906668509}{set\+\_\+field\+\_\+delimiter()} to specify how to turn lines into fields.
