Interpolate between two points. 

Interpolate between two different points in a trajectory. At t \texorpdfstring{$<$}{<}= 0 you\textquotesingle{}ll get the first point back. At t \texorpdfstring{$>$}{>}= 1 you\textquotesingle{}ll get the second point. At any value in between you\textquotesingle{}ll get a combination of the two. Coordinates and timestamp will be interpolated linearly. Numeric properties will be interpolated linearly. String properties will be taken from the first point if t \texorpdfstring{$<$}{<}= 0.\+5 and the second point otherwise.
