\doxysubsection{Detailed Description}
Interpolate between two property maps. 

Interpolation is well-\/defined for numbers and for timestamps. For strings, we choose the first string for {\ttfamily t \texorpdfstring{$<$}{<}= 0.\+5} and the second string for {\ttfamily t \texorpdfstring{$>$}{>} 0.\+5}.
