\doxysubsection{Detailed Description}
\subsubsection*{template$<$std\+::size\+\_\+t Dimension$>$\newline
class tracktable\+::\+Point\+Base$<$ Dimension $>$}
Base class for all points in Tracktable. 

This class defines a point independent of the number of coordinates or the data type.

\begin{DoxyNote}{Note}
You will not use this directly. Instead, you\textquotesingle{}ll use one of the coordinate-\/specific versions like Point\+Base\+Cartesian or Point\+Base\+Lon\+Lat.
\end{DoxyNote}
\doxylink{classtracktable_1_1_point_base}{Point\+Base} and all of its subclasses will be registered with {\ttfamily boost\+::geometry} so that you can use all of the generic geometry algorithms.
