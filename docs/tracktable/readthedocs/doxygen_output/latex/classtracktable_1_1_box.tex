Axis-\/aligned bounding box. 

This class is a thin wrapper around {\ttfamily boost\+::geometry\+::model\+::box} that holds two points representing opposite corners.

\begin{DoxyWarning}{Warning}
Don\textquotesingle{}t instantiate this class directly. Instead, use {\ttfamily "{}tracktable\+::domain\+::$<$name$>$\+::box\+\_\+type"{}} so that you get the correct distance functions.
