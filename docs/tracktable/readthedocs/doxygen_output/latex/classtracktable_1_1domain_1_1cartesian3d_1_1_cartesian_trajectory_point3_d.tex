\doxysubsection{Detailed Description}
\doxylink{classtracktable_1_1_trajectory}{Trajectory} point in flat 3D space. 

This class defines a point in 3D Euclidean space along with an object ID, timestamp and named properties. Units have no real-\/world interpretation and (unlike the surface of the globe) space is isotropic.

Use this class instead of instantiating the {\ttfamily \doxylink{classtracktable_1_1_trajectory_point}{Trajectory\+Point}} template yourself.
