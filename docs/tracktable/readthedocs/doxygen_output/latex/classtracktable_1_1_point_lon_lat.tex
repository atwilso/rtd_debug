\doxysubsection{Detailed Description}
2D point on a sphere 

This class specializes \doxylink{classtracktable_1_1_point_base}{Point\+Base} to use {\ttfamily boost\+::geometry} in a spherical-\/equatorial coordinate system -- the familiar longitude/latitude mapping onto a sphere.
