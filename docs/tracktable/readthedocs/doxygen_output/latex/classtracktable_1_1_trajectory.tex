\doxysubsection{Detailed Description}
\subsubsection*{template$<$class PointT$>$\newline
class tracktable\+::\+Trajectory$<$ Point\+T $>$}
Ordered sequence of points. 

This class is the heart of most of what Tracktable does. It implements an ordered sequence of \doxylink{classtracktable_1_1_trajectory_point}{Trajectory\+Point} objects, each of which has an ID, coordinates and a timestamp. Those compose a trajectory.

We provide accessors so that you can treat a \doxylink{classtracktable_1_1_trajectory}{Trajectory} as if it were a std\+::vector.
