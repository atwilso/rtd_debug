\subsubsection*{template$<$typename Uniform\+Random\+Number\+Generator = boost\+::random\+::mt19937$>$\newline
class tracktable\+::\+Boost\+Random\+UUIDGenerator$<$ Uniform\+Random\+Number\+Generator $>$}
Wraps a boost uuid random generator to act as a Random\+UUIDGenerator. 

The boost random number generator is templated to use various random number generation mechanisms. The default boost implementation is the mt19937 which is also the default template type of this class.

However, any random number generator suitable for the boost classes can be used with this thin wrapper template.

To create an instance of this class, use the various static {\ttfamily \doxylink{classtracktable_1_1_boost_random_u_u_i_d_generator_ae650f39971837126d734a93a3fcfa6eb}{create()}} methods that mimic the available boost random generator constructors.
