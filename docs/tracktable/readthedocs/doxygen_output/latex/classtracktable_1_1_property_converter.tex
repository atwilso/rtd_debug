\doxysubsection{Detailed Description}
\doxylink{classtracktable_1_1_property_converter}{Property\+Converter} -\/ Property\+ValueT to/from string. 

We need to be able to convert strings to Property\+ValueT instances and the reverse while using custom settings for decimal precision and time format. There are enough differences in the way locales are implemented across platforms that I insist on encapsulating it in this class. We will use this in readers and writers.
