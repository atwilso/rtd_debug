\doxysubsection{Detailed Description}
\subsubsection*{template$<$class StringT = std\+::string$>$\newline
class tracktable\+::\+Line\+Reader$<$ String\+T $>$}
Read data from a stream one line at a time. 

This class is an adapter that takes a {\ttfamily std\+::istream} as input and provides an Input\+Iterator that loops over the lines of the stream. In its current version it delegates to the {\ttfamily getline()} function to determine when a line has ended.

The behavior is meant to be the same as the Python idiom {\ttfamily for line in file}.

We do not expect you to instantiate this class directly. Instead, it shows up as part of {\ttfamily \doxylink{classtracktable_1_1_point_reader}{Point\+Reader}} where you put a stream in one end and get points out the other end.
