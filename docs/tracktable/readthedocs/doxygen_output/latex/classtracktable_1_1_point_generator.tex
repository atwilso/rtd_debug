class tracktable\+::\+Point\+Generator$<$ Point\+\_\+\+T $>$}
Point Generator for terrestrial points The is a basic point generator used as a base for all others. 

It will just keep incrementing points for a stationary object forever. Each call to next will increment the timestamp of the returned point by the set interval
