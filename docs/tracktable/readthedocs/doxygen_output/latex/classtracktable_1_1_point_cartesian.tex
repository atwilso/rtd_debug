class tracktable\+::\+Point\+Cartesian$<$ Dimension $>$}
N-\/dimensional point in Cartesian space. 

This specializes \doxylink{classtracktable_1_1_point_base}{Point\+Base} to exist in a Cartesian coordinate system and be usable with {\ttfamily boost\+::geometry}. You must still instantiate it explicitly with the number of dimensions.
