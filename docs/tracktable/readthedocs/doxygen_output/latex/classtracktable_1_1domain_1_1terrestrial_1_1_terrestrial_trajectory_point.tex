\begin{Indent}\textbf{ Earth Centered Earth Fixed}\par
{\em This group of functions is useful for converting longitude and latitude to a cartesian point.

A static method is provided to convert any arbitrary point. {\ttfamily ECEF\+\_\+from\+\_\+feet} will be the most common use case. Multiple functions will throw an exception if altitude is not present. To bypass exception throws, use {\ttfamily ECEF(ratio,alt\+String)} with the appropriate ratio to convert to km and the name of the property you are expecting to find altitude in. \begin{DoxySeeAlso}{See also}
