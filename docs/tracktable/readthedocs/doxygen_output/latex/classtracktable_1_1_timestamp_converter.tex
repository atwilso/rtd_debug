\doxysubsection{Detailed Description}
\doxylink{classtracktable_1_1_timestamp_converter}{Timestamp\+Converter} -\/ timestamp to/from string. 

We need to be able to convert strings to {\ttfamily \doxylink{namespacetracktable_a682e5fb98918f65770834e07ff51d21c}{tracktable\+::\+Timestamp}} instances and the reverse while using custom time input/output facets. There are enough differences in the way locales are implemented across platforms that I insist on encapsulating it in this class. We will use this in readers and writers.
